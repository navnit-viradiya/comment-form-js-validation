== Description ==
This plugin use for wordpress comments js validation to the comment form. Option for set your own validation message.

= Features =
* Easy to install
* Custom validation message

== How to Use ==
Only need to activate the plugin.

1. Go to Plugins > Add New. 
2. Searh \"comment form js validation\"
3. install and activate the WordPress Plugin.
